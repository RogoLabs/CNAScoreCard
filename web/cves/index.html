About EAS Scoring

The Enhanced Aggregate Scoring (EAS) methodology provides a 100-point, multi-dimensional assessment of each CVE record. EAS evaluates the following six dimensions:

  • Foundational Completeness (30 points): Product identification and high-quality vulnerability descriptions
  • Root Cause Analysis (10 points): CWE classifications and technical depth
  • Software Identification (10 points): CPE identifiers for affected products
  • Severity Context (25 points): CVSS scores and impact information
  • Actionable Intelligence (20 points): Solutions, patches, and references
  • Data Format Precision (5 points): Structured, machine-readable content

Each CVE is scored based on the presence and quality of these elements, enabling objective, data-driven comparisons across CNAs and disclosures.

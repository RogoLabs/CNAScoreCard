About EAS Scoring

The Enhanced Aggregate Scoring (EAS) methodology evaluates CVE records across six key dimensions:

  • Foundational Completeness (30 points): Basic information quality and completeness
  • Root Cause Analysis (10 points): CWE classifications and technical depth
  • Software Identification (10 points): CPE identifiers for affected products
  • Severity Context (25 points): CVSS scores and impact information
  • Actionable Intelligence (20 points): Solutions, patches, and references
  • Data Format Precision (5 points): Structured, machine-readable content

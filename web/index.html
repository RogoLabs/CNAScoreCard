CNA ScoreCard

A comprehensive evaluation platform that scores CVE Numbering Authorities (CNAs) based on the quality and completeness of their vulnerability disclosures using the Enhanced Aggregate Scoring (EAS) methodology.

What We Measure

🏗️ Foundational Completeness

Product identification, version details, and clear vulnerability descriptions

🔍 Root Cause Analysis

CWE classifications that help developers understand vulnerability patterns

⚡ Severity Context

CVSS scores and threat metrics for proper risk assessment

🎯 Actionable Intelligence

References, exploits, and VEX data for immediate security response

🆔 Software Identification (CPE)

Presence of valid CPE identifiers for affected products, enabling precise software targeting and automation

Live Data: Automatically updated every 6 hours from the official CVE database • Time Range: Last 6 months of CVE publications • Methodology: Enhanced Aggregate Scoring (EAS) with 100-point scale
